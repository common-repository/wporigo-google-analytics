WPOrigo Google Analytics Plugin is a very simple, but very useful plugin for WordPress.

== Description ==

This plugin add your Google Analytics code to your WordPress site.

== Installation ==

1. Install WPOrigo Google Analytics Plugin either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu on the WordPress Admin Dashboard
3. Add the Google Analytics code:  Admin Dashboard -> WPOrigo GA -> WPOrigo GA Options
